# README

faker.jsとjson-server.jsを使用して大量のモックデータを作成するツールです

バックエンドが出来上がってない状態でフロントエンドの開発をする際に仮のAPIを作成できます


## 起動方法
```
npm i

// 変更反映
npm run build

// APIサーバ起動
json-server --watch db.json --port 3002

```
